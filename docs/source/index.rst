Textplainer: Explain text contributions to ML Models
====================================================

Textplainer is a package and command line application that will
permit you to query a trained machine learning model
to discover the qualities of your text fields that are contributing
to predictive outcomes. These explanations will be provided as an encoded
string representation that can then be visualised in a range of methods.
